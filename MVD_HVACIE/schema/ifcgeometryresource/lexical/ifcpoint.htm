8.4.3.6 IfcPoint

Point
Punkt

The IfcPoint is the abstract generalisation of all point representations within a Cartesian coordinate system.

NOTE  Definition according to ISO/CD 10303-42:1992
A point is a location in some real Cartesian coordinate space Rm, for m = 1, 2 or 3.
NOTE  Entity adapted from point in ISO 10303-42.
HISTORY  New entity in IFC1.5
